A dead-simple just-add-water css library.

This page gives a few example usages. It uses Montserrat for headers, Open Sans for body text, and Courier Prime for links and code.

You can use it by putting this in the head of your page:

<link rel="stylesheet" href="https://unpkg.com/blocks.css/dist/blocks.min.css"/>

Examples

Here are some example usages of the classes found in the stylesheet.

This is a regular card. Use the class lite-card to style it like this.

This card has a hover effect! Use the class lite-hover to style it like this. It also has a button which looks pretty cool :) Style it by adding lite-button to the div or input. You can make it round by adding lite-round.

Add a shadow by using the class lite-shadow. This card also has some cool pill-style list items. Use them by adding the lite-hl class to your ul.
