A <2kB just-add-water CSS library.

This page gives a few example usages. It only adds fonts (Roboto for headers and body text, and Courier Prime for links and code) and the rest is pure lite.css.

You can use it by putting this in the head of your page:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/jackyzha0/lite.css@v1.2/src/lite.min.css"/>

Examples

Here are some example usages of the classes found in the stylesheet.

This is a regular card. Use the class lt-card to style it like this.

This card has a hover effect! Use the class lt-hover to style it like this. It also has a button which looks pretty cool :) Style it by adding lt-button to the div or input. You can make it round by adding lt-round.

Add a shadow by using the class lt-shadow. This card also has some cool pill-style list items. Use them by adding the lt-hl class to your ul.

Here's a picture of me! You can remove margins and padding by adding lt-clear. You can horizontally centre elements by adding lt-centre to its parent. If you're feeling spicy, you can also use flexboxes by adding either lt-flex-row or lt-flex-col to the parent div. Then, for each of the children, you can specify the flex basis. There are some default ones that you can use right away like lt-flex-50, lt-flex-33, lt-flex-66, and lt-flex-25. If you want to horizontally centre some text (or divs), wrap its parent with lt-flex-col lt-flex-centre.
